  • MotherEffin HTML5 Site : ASP.NET MVC 3 site template available as a Visual Studio Extension with OpenID.
  • django-html5-boilerplate: A Django project starting template with html5-boilerplate integrated.
  • django-cms-html5-1140px-boilerplate: A django-cms project starting template with html5-boilerplate integrated
  • MODX Revolution „HTML5Boilerplate Template Basis“ with control.js
  • Rails-3-HTML5-Boilerplate : Rails 3 implementation of HTML5 Boilerplate
  • h5bp-rails : Rails 3.1+ application template (rails new appname -m template) that HTML5 Boilerplate-ifies new Rails projects
  • Rails-HTML5-Boilerplate-Template : Rails 3.0.x application template (rails new appname -m template) that HTML5 BoilerPlate-ifies new Rails projects.
  • Sinatra-HTML5-Boilerplate : Get started with HTML5 Boilerplate on Sinatra, includes Cucumber and Capybara.
  • Flask-Boilerplate : Flask skeleton for Apache/mod_wsgi deployment
  • Flask Engine : Flask skeleton for Google App Engine with html5 boilerplate templating
  • Stacey CMS - HTML5Boilerplate Edition : joins staceyapp and html5-boilerplate
  • WordPress CMS - Integrating the HTML5 Boilerplate (including Build Script) with WordPress : Tutorials on adjusting the HTML5 Boilerplate to fit your WordPress theme.
  • Codeigniter-Boilerplate : Codeigniter-Boilerplate helps Codeigniter developers to start a new project in just a few minutes thanks to a simple template based on HTML5 Boilerplate.
  • CakePHP BakingPlate : Boost your CakePHP baking with a collection of community best-practices and built with HTML5BoilerPlate
  • Zend Frameword + HTML 5 Boilerplate : Bringing the awesome-ness of HTML5 Boilerplate and the Zend Framework together!
  • HTML5 Boilerplate Plus+ : Bulletproof HTML5 polyfilling addition for non-modern browsers
  • rails-boilerplate : Rails (>= 3.1) gem that helps you at starting out with the html5 boilerplate very easy, it provides a generator for the layout, and provides the javascripts and stylesheets as assets.
