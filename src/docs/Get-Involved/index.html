Get Involved

♥ H5BP and want to get involved? There are plenty of ways you can help!

Reporting issues

Use the GitHub issue tracker. Where possible, please include a reduced test case because it really helps make troubleshooting easier.

Please try to be as detailed as possible in your report too. What is your environment? What steps will reproduce the issue? What browser(s) and OS experience the problem?

Contributing code

Submit a pull request

Please thoroughly test any code before submitting a pull request, and it would be great if you could also try and follow the existing conventions used in the boilerplate (2 space indentation, consistent use of single and double quotes, etc).

Documentation

Our docs are used by nearly all of our users, getting to know the rationale behind certain decisions and also learning how to use the parts of the boilerplate effectively. We'd love help adding detail to the docs; we're looking for writers and also caretakers to watch incoming contributions to them.

These pages could do with some love

  • Home - better organization and links to orphan pages
  • Build Script - better organization, culling old information, and making sure all project customizable properties are documented, and all the different build options are documented.
  • Offline - needs more detail on creating a manifest

Build Tool

We are planning to have a Rake version of the Build Script, for the latest info, visit issue: https://github.com/h5bp/html5-boilerplate/issues/516

Translation

To help with translation, visit html5boilerplate-site, it's now in 11 languages and counting!

The wiki section has the current translation status, tips and tricks on how to translate.

Discussion

Where to find people of H5BP if you have something to discuss? Most of us can be found in freenode IRC channel #HTML5 and we also have a Google Group. We love your help answering other people's questions on the google group. There is also regular threads about H5BP on StackOverflow that you can jump in on.
