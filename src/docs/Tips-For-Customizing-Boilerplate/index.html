Using another JavaScript Framework

HTML5Boilerplate is pretty framework agnostic, but includes jQuery because it is a popular library that fits the needs of most people. If you intend to use another library, look in the FAQs. If you would like to use frameworks in addition to jQuery, consider including them in plugins.js.

We like to split our CSS

  • Just note that every extra resource you add to style your page, it takes longer to load it.
  • Make sure you move the rules following these "/* Primary Styles \n Author: */ into your new stylesheet. Media Queries always need to be after your styles and cannot be defined before your styles.
  • If you would like to update your Boilerplate CSS to a new version, remember to update your own stylesheet for any changes in the rules for media queries.
  • Make sure you call the default stylesheet first before calling your new stylesheet.
