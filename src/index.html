{% blocktrans %}A rock-solid default for HTML5 awesome.{% endblocktrans %}

{% blocktrans %}I'm new here: plz explain why it's so awesome, first.{% endblocktrans %}

{% blocktrans %}HTML5 Boilerplate is the professional badass's base HTML/CSS/JS template for a fast, robust and future-safe site.{% endblocktrans %}

{% blocktrans %}After more than three years in iterative development, you get the best of the best practices baked in: cross-browser normalization, performance optimizations, even optional features like cross-domain Ajax and Flash. A starter apache .htaccess config file hooks you the eff up with caching rules and preps your site to serve HTML5 video, use @font-face, and get your gzip zipple on.{% endblocktrans %}

{% blocktrans %}Boilerplate is not a framework, nor does it prescribe any philosophy of development, it's just got some tricks to get your project off the ground quickly and right-footed.{% endblocktrans %}

{% blocktrans %}HTML5 Boilerplate is 3.0!{% endblocktrans %}

{% blocktrans %}Monday, Feb 6th, 2012{% endblocktrans %}

{% blocktrans %}The key feature of this update is making Boilerplate smaller. Most of the changes have been working towards that. We have significant work done to our build script, so much so that we thought it deserved its own repository. Meanwhile, here is what's new.{% endblocktrans %}

{% blocktrans %}Changelog{% endblocktrans %}

{% blocktrans %}3.0 gets some major updates Here are all the commits since last release.{% endblocktrans %}

{% blocktrans %}Should you upgrade existing sites? Short answer: nah, you're good.{% endblocktrans %}

{% blocktrans %}Significant changes{% endblocktrans %}

  • {% blocktrans %}Removed a lot of folders: /demos, /test, and /js/mylibs.{% endblocktrans %}
  • {% blocktrans %}Build script is now a separate project{% endblocktrans %}, {% blocktrans %}Read about how to get and use the build script{% endblocktrans %}
  • {% blocktrans %}Respond.js is no longer available by default. {% endblocktrans %}

index.html

  • {% blocktrans %} Chromeframe no longer applied via script. IE6 users are served links to install Chromeframe or visit Browsehappy to choose an alternative browser. {% endblocktrans %}
  • {% blocktrans %}Meta viewport updated to prevent iOS scaling bugs. {% endblocktrans %}
  • {% blocktrans %}html IE classes now target lower than a specific version rather than explicitly specified versions. {% endblocktrans %}
  • {% blocktrans %}Switch back to the optimized asynchronous Google Analytics snippet instead of using Modernizr.load. {% endblocktrans %}
  • {% blocktrans %}Drop defer from all scripts to avoid legacy IE bugs.{% endblocktrans %}
  • {% blocktrans %}Design of the 404 has been improved.{% endblocktrans %}

style­.css

  • {% blocktrans %}Removed default font sizes and line heights to provide a much better robust default. {% endblocktrans %}
  • {% blocktrans %}Additions made to normalize.css and to provide better default typography.{% endblocktrans %}
  • {% blocktrans %}More robust .ir helper class.{% endblocktrans %}
  • {% blocktrans %}No more forced scrollbars, this impedes projects like jQuery UI, and others in calculating widths.{% endblocktrans %}

{% blocktrans %}Project{% endblocktrans %}

  • {% blocktrans %}Updated to Modernizr 2.5{% endblocktrans %}
  • {% blocktrans %}Updated to jQuery 1.7.1{% endblocktrans %}
  • {% blocktrans %}Improved .htaccess{% endblocktrans %}
